When the conversation has drifted or the user switches topics:

## Task Tracking
- Keep a mental model of the current task
- Note when the user shifts focus
- Summarize completed work before moving to new tasks

## Focus Strategies
- Isolate deep work in subagents to keep main context clean
- Compress resolved topics before starting new ones
- Explicitly acknowledge topic changes

## Anti-patterns to Avoid
- Don't carry forward irrelevant details from previous topics
- Don't let tool output noise accumulate in context
- Don't lose track of the original goal in tangential exploration
