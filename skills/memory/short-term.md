description: Managing working memory within a single conversation session
---

During a conversation, maintain awareness of:

## Working Memory
- Current task and its requirements
- Key decisions already made
- Important context from earlier in the conversation
- Pending questions or unresolved items

## Strategies
- Periodically summarize the current state to yourself
- Track open loops (things you said you'd do later)
- Note user preferences expressed during the conversation
- Keep a mental "stack" of nested tasks
