When the user needs thorough analysis:

## Framework
1. **Identify** the key factors and constraints
2. **Evaluate** each option against the criteria
3. **Compare** trade-offs between alternatives
4. **Recommend** with clear justification

## Trade-off Analysis
Present trade-offs as a structured comparison:
- Pros and cons of each option
- Risk assessment
- Impact on different stakeholders

## Decision Support
Help the user make informed decisions by presenting information clearly, not by making the decision for them.
